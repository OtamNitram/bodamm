# Figma Images - Manual Export Required

## Task T010: Extract images from Figma

**Figma File**: https://www.figma.com/design/qwrm6VDQhaEODfbkZfQ6Kk/Casamiento

### Required Exports

Export the following frames in **WebP format** at 1x, 2x, and 3x resolutions:

1. **Home Web** (node 10:197)

   - Export as: `home-web@1x.webp`, `home-web@2x.webp`, `home-web@3x.webp`

2. **Home Tablet** (node 11:1176)

   - Export as: `home-tablet@1x.webp`, `home-tablet@2x.webp`, `home-tablet@3x.webp`

3. **Home Mobile** (node 11:1017)
   - Export as: `home-mobile@1x.webp`, `home-mobile@2x.webp`, `home-mobile@3x.webp`

### Export Instructions

1. Open Figma file
2. Select each frame
3. In the right panel, click "Export"
4. Set format to WebP
5. Add 1x, 2x, 3x export sizes
6. Export and save to this directory

Extract any other images used in the design (backgrounds, decorative elements, etc.) and save them here with descriptive names.
