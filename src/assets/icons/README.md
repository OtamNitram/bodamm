# Figma Icons - Manual Export Required

## Task T011: Extract icons from Figma

**Figma File**: https://www.figma.com/design/qwrm6VDQhaEODfbkZfQ6Kk/Casamiento

### Required Exports

Export all icons used in the design in **SVG format**:

- Navigation icons (if any)
- Social media icons (WhatsApp, Telegram, Spotify)
- Decorative icons
- Any other UI icons

### Export Instructions

1. Open Figma file
2. Select each icon
3. In the right panel, click "Export"
4. Set format to SVG
5. Export and save to this directory with descriptive names (e.g., `whatsapp.svg`, `telegram.svg`, `spotify.svg`)

### Naming Convention

Use lowercase with hyphens:

- `whatsapp.svg`
- `telegram.svg`
- `spotify.svg`
- `menu-icon.svg`
- etc.
